Kubus 3D Interaktif

Aplikasi kubus 3D yang dapat digerakkan dengan keyboard. Gunakan tombol panah untuk memutar kubus, tombol W/A/S/D untuk menggerakkan kamera, dan tombol +/- untuk zoom.

Cara Menyimpan ke GitHub

  1. Buat repositori baru di GitHub
  2. Salin kode HTML ini dan buat file dengan nama index.html
  3. Unggah file tersebut ke repositori GitHub Anda
  4. Aktifkan GitHub Pages pada pengaturan repositori
  5. Akses aplikasi melalui URL GitHub Pages yang disediakan

Unduh Kode untuk GitHub

Salin seluruh kode halaman ini dan simpan sebagai file HTML, lalu unggah ke GitHub.
